###################################### Tetris Game #################################################
#
# Please read https://en.wikipedia.org/wiki/Tetris
#
# Here is an online tetris game if you haven't played tetris before:
# http://tetris.com/play-tetris/
#
# Tetrimino: A piece in the game. We have 7 tetriminos in total:
#
# Tetrimino_I, Tetrimino_J, Tetrimino_L, Tetrimino_O, Tetrimino_S,
# Tetrimino_T, Tetrimino_Z
#
# All 7 tetriminos have 4 squares in them. Please see this visualization:
# https://en.wikipedia.org/wiki/Tetris#Colors_of_Tetriminos
#
# See images/@Overview.png to see the big picture of the game.
#
# Watch the video: https://youtu.be/Ts6EaEqIDnk
#
# CONTROLS: Left-arrow: Move left
#           Right-arrow: Move right
#           Up-arrow: Rotate
#           Down-arrow: Move down faster
#           Space key: Move down very fast
#
